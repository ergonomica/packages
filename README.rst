packages
========

The central Ergonomica package list.

Listing Format
--------------

The format is:

.. code::
   
   packagename.py(semver_version)
   package_description
   package_file_url

For example,

.. code::

   vortex.py(1.0.0)
   Jump to directories.
   https://raw.githubusercontent.com/ergonomica/package-vortex/master/vortex.py
